Welcome to Kitsune!

Kitsune helps you find the closest semantic mappings for your data using advanced embeddings generated by large language models (LLMs) and pre-trained language models (PLMs). It supports various standards and ontologies to ensure precise and meaningful matches.

To get started, please select your preferred search method:

Query Search - Find semantic mappings by submitting a custom query.

Dictionary Search - Upload a dictionary of variables to retrieve semantic matches.

Select a method to proceed and discover the best semantic matches across various ontologies and standards.

If you encounter any issues or would like to report a bug, you can do so via the GitHub link in the top-right corner.
